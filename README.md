# 🚐 VanLife  

> A full-stack simulated van rental platform where users can explore, view, and manage camper vans for road trips.  

---

## 🪞 Overview  
VanLife is a React-based web application that allows users to browse and rent travel vans while also enabling hosts to manage their listed vehicles.  

It solves the problem of finding and managing van rentals in a simple, intuitive way. The app is built for travelers who love road trips and hosts who want to list their vans.  

What makes it unique is the **host dashboard experience**, protected routes, and a fully mocked backend using MirageJS—giving it a real-world feel without an actual server.  

---

## 🚀 Features  
✨ Key things your app can do:  

- 🚐 Browse all available vans with details  
- 🔍 View individual van information (price, type, description)  
- 🔐 Authentication with protected routes  
- 🧑‍💼 Host dashboard to manage listed vans  
- 📊 Income & reviews sections for hosts  
- 🔄 Nested routing with dynamic routes  
- ⚡ Mock API using MirageJS
